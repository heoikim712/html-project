Adding Links=

Adding links is kind of comparable to a spider shooting out webs to other surfaces to navigate to other surfaces. By adding links in our HTML file, we can
give the audience the ability to navigate into other websites or other parts of our page.


In the image, the usage of an <a> tag (anchor)is used. The anchor tag is used to add links to navigate to different places in the web. This includes
Relative Links that links to different places on your webpage, and Absolute Links that link to other webpages. Absolute Links are links that contain the
full and explicit location of a resource, starting from the root directory of the webdite or the server.

Types of Links I can Add=

1.) Navigational Links=

Here are the tags you'll need to know in order to do this. The <nav> and <section> tags. To add the navigational link on the page, use the <nav> element
and add an <a> tag and put hashtag and the section id's value for the value of the href for the link to "link" to the section of the content you want the user to
navigate to.

You may have noticed that I've mentioned something about a section id, and in this paragraph I'll explain more upon that topic. The Section Id is used to
indicate the link's destination, that's why we need the Id and the value of href to be the exact same value, so that the link will know to navigate to that same
exact to send the user to. This way, the user will have access to links to travel to different places of your webpage or any other webpage (with the Absolute
Link provided).

Target Attribute=

There is an Attribute that lets you change the way the link is opened. You simply have to add the Taget Attribute and then put the specific value to make the
link open in a certain way.
Here are all the possible values for the Target Attribute=

_blank
Opens the linked page/resource in a new browser window or tab.
_parent
Opens the linked page/resource in the parent frame or window, if applicable.
_self
Opens the linked page/resource in the same window/tab.
_parent
Opens the linked page/resource in the parent frame or window, if applicable.
_top
Opens the linked page/resource in the top-level browsing context, replacing all frames if applicable.
_framename
Opens the linked page/resource in a specific named frame or window, where "framename" is the name attribute of the target frame/window.
