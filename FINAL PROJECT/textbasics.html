Text Basics

In this page, I'll be teaching you about the basics of texts in an HTML file and how to modify or in other word,
format it. Cheers, and enjoy today's lesson.

Text Formatting

There are ways how to format your texts in HTML. In order to do that, you need to know some elements. With
these elements, you'll be able to change the size and underline, highlight, italicize, and make your texts in bold.

Here are a list of elements that'll format your texts. However, do not forget that there will be more elements that
can be used, I'll provide only some elements just to keep this lesson as short as possible.

  1. <u>= underlines text.
  2. <mark>= highlights text.
  3. <em>= italicizes text
  4. <strong>= bold text
  5. <style>= (needs attributes to "decorate" texts)
  6. <b>= bold text (used for semantic purposes)
  7. <i>= italicized text (used for semantic meaning)

And these are some of the elements that format texts in an HTML archive. We can see that these elements give
the texts a different look by marking them in different ways, thus making them appear easier to the human eye.
We can also see that there are formatting elements that give semantic meaning like the <b> and <i> tags.
