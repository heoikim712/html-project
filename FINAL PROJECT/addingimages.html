Adding Images=

Your website would look extremely boring without any images. Imagine if Amazon had no images! If that was true, you wouldn't see any images of
the products you are willing to purchase. That would be terrible! In this lesson, I'll teach you guys how to add Images into your websites to make
them more interesting and interactive.

Steps=

Use <img> Element=

First, add the <img> tag and you'll see that VSC will automaticly add the source and alternate attributes for you. In the source attribute, you have to
add the relative link or an absolute link of the image you would like to add. Then, in the alt attribute, you'll need to type a short sentences (few words)
that will apear just in case if there's a problem in the process of loading the image. This way, even if the image doesn't load, the audience will have
an idea of what the image is about. Next, I'll show you guys other attributes that you can add to your <img> tag.

Title Attribute=
Another attribute you can add is the title attribute. Whatever text you add for the value of the attribute will apear if the user puts their cursor
on top of the image.

Width and Height Attribute=
As the name suggests, this attribute is used to modify the dimention of the image. The width modifies the width, and the heiht modifies the height.
When you see the dimention of and image, you'll see something like this, 200 x 400. The first digit (200) signifies the width and the second digit
(400) tells you the height. However, modifying your image's dimention with CSS is more recomended, however, doing it with just HTML is just
as fine with CSS.
