For today's lesson, I'll teach you guys a way to add Tables in an HTML5 file. With this skill you can make things like=

  1. Schedule Tables
  2. Sales Tables

Uses=

With the <table> tag, we can create a table and add table datas and create some very interesting tables to show the audience tabular data.

Elements we'll Use=

Obviously, to create a table, we'll need some tags in order to do that. Here are the Elements and Attributes you'll need to know.

Elements we'll Use
Elements=
<table>
<tr>
<th>
<td>
<caption>
Attributes=
Colspan
Rowspan
border
Scope

Elements and Attributes Explained=

The <table> tag creates the table format. The <tr> tag creates a table row and inside that tag you'll nest either a <th> or <td> tag. The th stands for table head and the td stands for table data. Finally, the <caption> tag is used for adding a caption for the table.

In this paragraph, I'll explain all the Attributes that I used. The rowspan and colspan are used to modify the <tr> to take up 2 rows or 2 columns (you can change the number depending on your case). I used the border attribute to make the table actually look like a table (physically), and I added the value of 6px for the thickness of the lines. You "should" use CSS but this Attribute let's me create lines for the table in a much simpler way. And finally, the Attribute scope is used in the context of table headers (<th>) to define the scope of a header cell. It is primarily used for accessibility purposes to indicate whether a header cell applies to a row, a column, or a group of rows or columns within a data table.
