Adding Videos and Audios in an HTML5 file=

You can add video and audio files in an HTML5 files. A great example of video files existing in an HTML file is YOUTUBE. I mean, imagine if there wasn't a way to add a video file inside a website. Websites like Youtube, Netflix, and Amazon Prime Video


ADDING VIDEOS=

Adding videos are dead easy. To put this into perspective, the only tag you need to know is the <video> tag and some other attributes.

A basic video tag would look something like this=

I will now explain all the attributes that were inside the image I just displayed to you. The src attribute is used for selecting the video file. The controls and loop are boolean attributes. This means that the attribute doesn't have any values, and the addition of the attribute inplies a "true" or "on" state. They are often used for enabling or disabling certain features or behaviours in HTML Elements.

After the <video> tag, you may have noticed that I nested some <source> tags. Let me tell you the purporse of this. The point of adding this tag is just in case your browser doesn't support the video format that you added. So I added the .webM format just in case the browser the user uses doesn't support the .mp4 video file. You can add even more <source> tags to make the video more accessable to more people that use browsers that support different types of video formats.

ADDING AUDIOS

Just like adding video files, adding audios in an HTML5 file is an easy task. In this case, rather than using the <video> tag, you'll need to use the <audio> tag

The attributes that are in an <audio> tag are basicly the same as the attributes inside a <video> tag. So you'll use the "controls" "loop" "autoplay" and many more. You'll also use the <source> tags, and add the audio file formats so that other browser that doesn't support the audio file format that you added, doesn't support.
